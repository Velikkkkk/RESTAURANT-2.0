# Brief Summary
## The project aims to:
- Scanning all restaurant entries from a pre-compiled file and transferring this information into an ArrayList. Selecting a single random restaurant—along with its associated properties—and printing the details of that randomly selected restaurant.
- The "main" component serves as a data distribution system. First, it retrieves the data that has been scanned by the scanner, and then it establishes the specific locations where this data will subsequently be stored. Next, the data is formatted into a template suitable for subsequent printing.
- The information utilized for this project was pre-compiled through a survey conducted among the students and staff of the ACC.
